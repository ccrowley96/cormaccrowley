Queen's Library System

This is a Queen's library item checkout system built in java using object oriented programming techniques. The system allows items to be scanned from files or created by the user, and then checked out by customers of the library. Items and transactions are searchable and time-stamped. Checkout the all of the classes and source code at the Github link below.

Algorithm

  1. The item checkout system allows the user to create new items of type 'device' or 'book' as shown in the class heirarchy above. Items can also be created by reading file input.
  2. The system creates 'rentals' for each item allowing people to checkout textbooks, magazines, adaptors, and laptops.
  3. The system then adds the rentals to the library system as transactions time-stamped at the time of checkout.
  4. The system calculates total rental fees and total late fees based on the type of item, the time of checkout, and the item return date.
  5. The library system then calculates the total cost of all rented devices using: transactions.get(i).getItem().getDeviceRentalFees(); -- an example of polymorphism in Java.

Matrix Calculator

This is a Java matrix calculator I built. The purpose of the calculator is to perform matrix calculatations that are difficult, time consuming, and easy to mistake by hand. The code block below shows the inverse() method from the calculator.

Inverse Operation - Calculator

//Inverse Matrix -- Return inverse of matrices (1x1,2x2,3x3)
public Matrix inverse(){
Matrix newMat = new Matrix(this.m_rows,this.n_cols);

//check if square matrix
if(this.isSquare()){

//1x1 inverse
if(this.m_rows == 1){
	newMat.arr[0][0] = 1/this.arr[0][0];
}
//2x2 inverse
else if(this.m_rows == 2){
	double det = this.determinant();
	det = 1/det;
	newMat.arr[0][0] = det*this.arr[1][1];
	newMat.arr[1][1] = det*this.arr[0][0];
	newMat.arr[0][1] = -1*det*this.arr[0][1];
	newMat.arr[1][0] = -1*det*this.arr[1][0];
	
}
//3x3 inverse
else if(this.m_rows == 3){
	double det = this.determinant();
	if(det == 0){
		System.out.println("Determinant = 0, cannot find inverse");
		return newMat;
	}
	this.cofactorAndminor();
	Matrix temp = this.transpose();
	
	Matrix temp2 = temp.multiply(1/det);
	return temp2;
	
}
else{
	System.out.println("Problem encountered finding inverse");
	return newMat;
}
	
}else{
	System.out.println("Cannot find inverse of non-square matrix");
	return newMat;
}
return newMat;
}

Constructors

  1. Constructor to create a matrix given m and n
  2. Constructor to read a matrix from a user using terminal or popups
  3. Constructor to read a matrix from a .mat file. The file starts with the size of the matrix using two integers (m and n) separated by a comma.

Operations

  • Matrix add(Matrix m): Add this matrix to matrix m and return the result
  • Matrix subtract(Matrix m) : Subtract matrix m from this.matrix
  • Matrix multiply(Matrix m) : Multiply this.matrix and matrix m.
  • Matrix multiply(double x) : Multiply scalar x by this matrix.
  • Matrix divide(Matrix m): Matrix division of this.matrix on matrix m.
  • double determinant() : return the determinant of the current matrix
  • Matrix inverse() : return the inverse of the current matrix
  • boolean isSquare() : return true if the matrix is square
  • Matrix transpose() : return the transpose of the current matrix
  • Matrix identity(int s): Create an Identity square matrix of size s
  • void print(String filename): create a file and write the matrix in it.
  • String toString(): return a string representation of the matrix.
  • double get(int i, int j): get the value stored in i and j entry of the matrix
